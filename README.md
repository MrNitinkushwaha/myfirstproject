# Alumni Network Website

A fully functional alumni website frontend built with pure HTML, CSS, and JavaScript.

## Features

- **Responsive Design** - Works on all devices
- **Client-side Routing** - Single page application navigation
- **User Authentication** - Login/Register functionality
- **Alumni Directory** - Search and filter alumni
- **Events Calendar** - View and register for events
- **Job Board** - Browse and apply for jobs
- **News Section** - Stay updated with college news
- **User Profiles** - Personal dashboard

## Pages

1. **Home** - Landing page with features overview
2. **Directory** - Searchable alumni directory
3. **Events** - Upcoming events and registration
4. **Jobs** - Job opportunities from alumni
5. **News** - College news and updates
6. **Login** - User authentication
7. **Register** - New user registration
8. **Profile** - User profile dashboard

## Getting Started

1. Download all files maintaining the folder structure
2. Open `index.html` in a web browser
3. No server required - works directly in browser
